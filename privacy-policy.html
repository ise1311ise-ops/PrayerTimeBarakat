Здесь описано, какие данные обрабатывает приложение PrayerTimes Barakat, для чего они нужны, где хранятся и как вы можете управлять ими.

В текущей версии приложение не требует регистрации и хранит настройки в основном локально на устройстве. Если в будущем добавятся аналитика/реклама/аккаунты — политика будет обновлена.

1) Кто является оператором данных (контакты)

По вопросам конфиденциальности вы можете связаться с нами по email: support@barakat.app.

2) Какие данные обрабатывает приложение

2.1. Данные, которые предоставляет устройство/пользователь

  • Геолокация (примерная/точная — зависит от выданных разрешений): используется для определения города и расчёта времени молитв по координатам.
  • Настройки приложения, которые вы выбираете:
    • метод расчёта времени молитв;
    • корректировки времени молитв (если вы вручную меняете время);
    • язык интерфейса;
    • настройки уведомлений (за сколько минут предупреждать, режим «со звуком/без звука», выбранный рингтон).
  • Технические данные запроса при обращении к стороннему API (например, IP-адрес, время запроса и т.п.) — могут обрабатываться на стороне сервиса, к которому выполняется запрос.

2.2. Что приложение не собирает (в текущей версии)

  • не требует создания аккаунта;
  • не запрашивает и не собирает имя, номер телефона, email пользователя для идентификации;
  • не запрашивает доступ к контактам и медиафайлам;
  • не содержит встроенной рекламы и аналитики (при добавлении в будущем политика будет обновлена).

3) Для чего используются данные

  • определение города и получение времени молитв по координатам;
  • показ расписания и таймера до следующей молитвы;
  • создание и показ уведомлений о времени молитвы (в момент или заранее);
  • сохранение ваших настроек (язык, рингтон, режим уведомлений, корректировки времени).

4) Где и как хранятся данные

  • Настройки и кэш времени молитв сохраняются локально на вашем устройстве (в хранилище настроек приложения).
  • Мы не предоставляем сервер для хранения ваших персональных данных и не выгружаем их на наши серверы.

5) Передача данных третьим лицам и сторонние сервисы

Приложение использует сторонние сервисы для отдельных функций:

  • API времени молитв (AlAdhan) — приложение запрашивает расписание молитв по координатам через публичный API. При обращении сервис может получать технические данные запроса (например, IP-адрес) и переданные координаты.
  • Google Play Services (геолокация) — для получения местоположения может использоваться Fused Location Provider. Обработка данных происходит в рамках работы сервисов Google на устройстве.
Обработка данных этими сервисами регулируется их собственными политиками конфиденциальности.

6) Разрешения Android и зачем они нужны

  • INTERNET — запросы к API времени молитв.
  • ACCESS_COARSE_LOCATION / ACCESS_FINE_LOCATION — определение местоположения и корректный расчёт времени.
  • POST_NOTIFICATIONS — показ уведомлений (Android 13+).
  • SCHEDULE_EXACT_ALARM / USE_EXACT_ALARM — точные напоминания в нужное время.
  • RECEIVE_BOOT_COMPLETED — восстановление запланированных уведомлений после перезагрузки.
  • WAKE_LOCK — более надёжное срабатывание уведомлений/будильников.

Вы можете отключить доступ к геолокации и/или уведомлениям в настройках Android — приложение продолжит работать, но часть функций может быть ограничена.

7) Удаление данных

Данные хранятся локально, поэтому вы можете удалить их так:

  • удалить приложение (удалятся настройки и кэш);
  • или очистить данные приложения в настройках Android.

8) Безопасность

Мы применяем разумные меры для защиты данных на устройстве. Однако ни один способ хранения или передачи данных не может гарантировать абсолютную безопасность.

9) Дети

Приложение не предназначено специально для детей и не собирает данные, направленные на их идентификацию. Если вы считаете, что ребёнок предоставил персональные данные, свяжитесь с нами: support@barakat.app.

10) Права пользователей (включая GDPR/ЕЭЗ)

Если вы находитесь в ЕЭЗ/ЕС, у вас могут быть права на доступ, исправление и удаление данных. В текущей реализации данные хранятся локально — вы контролируете их через настройки устройства (см. раздел 7). По вопросам обращайтесь: support@barakat.app.

11) Изменения политики

Мы можем обновлять Политику конфиденциальности. Актуальная версия всегда будет доступна по ссылке, указанной в Google Play и/или в приложении.
